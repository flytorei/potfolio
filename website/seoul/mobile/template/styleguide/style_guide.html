  • IP주소신청시 이미 다른 IP에 등록된 랜카드주소(Ethernet address)로 신청하는 경우 IP 할당이 불허됩니다.

  • [실제 사용자]와 신청자가 다른 경우 반드시 기재해 주시기 바랍니다.
