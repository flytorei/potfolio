무선랜이용안내

이전

SNU WiFi

이용자

- 학내구성원, 방문자

사용방법

학내 구성원
  1. SNU-1st-Member 선택
  2. 브라우저 실행, 이용기기별 안내 확인 후 환경 설정
  3. SNU_ID/ PW 으로 로그인
  4. 이후 무선랜 목록 중 SNU-Member 이용
방문자
  1. SNU-1st-time 선택
  2. 임시 계정 생성 (이용시간 제한 : 4시간 )

SNU_U+Zone

이용자

- 모두 허용

사용방법

  1. 스마트폰 / 패드 / 노트북 암호 "lguplus100" 설정
  2. 설정 후 사용

※ 유료가입자는 L+zone4 이용

T Wifi zone

이용자

- 모두 허용

사용방법

  1. T wifi zone를 선택
  2. 절차에 따라 해당기기의 Mac등록
    (1개월간 인증 필요 없음, 등록시 휴대폰으로 인증번호 발송)

※ 유료가입자는 T wifi zone_secure 이용

KT WiFi

이용자

- 학내 구성원

사용방법

  1. 해당기기의 무선랜 목록 중 KT_SNU 선택
  2. SNU_ID/PW 로그인 (인증후 1년 인증 불필요, 단 6개월간 미사용시 인증필요)

※ 유료가입자는 ollehWiFi 이용
