검색 메뉴

아이피신청

이전

신청 및 조회

  • 신청하기
  • 조회하기

신청방법

  1. 인적정보, 위치 및 용도 입력
  2. 입력클릭, 여러개 신청시 [1]작성후 입력
  3. 신청클릭

주의사항

  • IP주소신청시 이미 다른 IP에 등록된 랜카드주소(Ethernet address)로 신청하는 경우 IP 할당이 불허됩니다.
  • 공유 등을 위해 특정 IP대역이 필요하시면 비고란에 적어주세요(예:147.46.100대역으로 할당해 주세요).
  • 책임자와 신청자의 이메일주소, 전화번호 정보를 정확히 입력해 주시기 바랍니다.
    (신청서에서의 수정은 해당 IP주소 정보에만 적용됩니다. 자동으로 보이는 값들은 인적DB에서 가져오는 것이므로 수정하시려면 my.snu.ac.kr 사이트 로그인 후 오른쪽 상단의 '마이페이지'에서 수정하시면 됩니다)
  • [실제 사용자]와 신청자가 다른 경우 반드시 기재해 주시기 바랍니다.

주소 할당 절차

  • IP주소 신청>IP주소 할당(메일 통보, 책임자에게 동의요청 메일 발송>책임자 할당 거부시 IP주소 강제 회수)
  • 관리자의 할당 처리 시간은 일과 중 매시 30분이며, 결과는 메일로 통보됩니다.
