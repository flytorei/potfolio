POLY는 학생의 연력, 학년
그리고 영어 능력에 따라 교육과정을
나눠 운영하고 있습니다.

학생의 영어 능력은, POLY가 직접 개발한 ‘영어 능력 진단 평가 시스템’인 ‘POLY’을 기반으로 분석하고 진단합니다. 5세 과정은 별도의 영어 능력 진단 평가가 없으며, 소정의 인터뷰로 대신합니다.

POLY가 직접 개발한 영어 능력 진단 평가 시스템

PLAT(POLY Language Aptitude Test)

POLY은 POLY의 영어교육 노하우를 집약해 개발한 영어 능력 진단 평가 시스템입니다.

그간 축적된 방대한 영어교육 데이터를 기반으로, 영어 능력 전반을 분석할 수 있는 체계를
구축했습니다. PALT은 POLY 교육과정의 시작이라 할 수 있습니다.

  • 입학상담

  • PLAT 예약,응시

  • PLAT 결과 확인

  • 학급배정
