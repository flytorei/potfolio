CARA’s Module

  • Vessel Schedule
  • Vessel Operation
  • Cargo Application
  • Allocation
  • Settlement
01
Schedule Module

Management vessel schedule from proforma schedule to actual schedule.

02
Benefit in Schedule Module

Providing schedule simulation for proforma and coastal schedule.

Recognizing carriers’ schedule included in e-mail or EDI automatically.

Analyzing precise on-time performance.

Included distance table between ports and ECA points.

03
Functionality in Schedule Module

Simulate proforma schedule in terms of service as well as cost, and finalize it to be used in generating long-range schedule.

Based on the generated long-range schedule, coastal schedule in each voyage to be updated and actualized through received e-mail or EDI from carriers.

Analyze on-performance for internal management as external evaluation.

01
Operation Module

Importing Baplie file and generating operation reports.

02
Benefit in Operation Module

Recognizing carriers’ bay plan information through included Baplie file in e-mail automatically.

Handling various private code in Baplie based on historical conversion pattern to improve recognition Baplie.

Providing summary of space/weight utilization for settlement under various joint operation or slot sales/purchase contracts.

03
Functionality in Operation Module

Generating container booking forecast based on container load list.

Importing Baplie files automatically via e-mail attachment.

Generating terminal departure report and regional departure report based on imported baplie file automatically.

Analyze terminal productivity performance for management.

01
Application Module

Handling special cargo applications between booking line and carrier.

02
Benefit in Application Module

Recognizing booking lines’ special cargo application included in e-mail or EDI automatically.

Validating the content of application in details to avoid irregularity in handling special cargo.

Included the latest IMDG Code DB.

03
Functionality in Application Module

Review in-bound applications from booking lines for loading DG/OOG/BB/RF/Special Stowage and change of destination.

Validate applications against various criteria including IMDG Code, regulations, restriction, certificate, and policy.

Response to booking line whether the application is acceptable or not.

01
Allocation Module

Setup agreed space/weight allocation for settlement.

02
Benefit in Allocation Module

Setting basic allocation easy way.

Roll out agreement to each voyage automatically.

Supporting various setup of allocation and sub-allocation for precise settlement.

03
Functionality in Allocation Module

Registering basic allocation and settlement agreement.

Rolling out basic allocation and price information to each voyage.

Updating allocation and price data for each voyage if it is required.

01
Settlement Module

Setup agreed space/weight allocation together.

02
Benefit in Settlement Module

Providing settlement configuration which able to generate settlement result in every contract condition.

Generating settlement details automatically based on settlement configuration.

Accurate settlement based on actual schedule, baplie, allocation data.

03
Functionality in Settlement Module

Setup target schedule.

Setup settlement item and price.

Setup settlement option.

Generating target schedule which will be included in the periodical settlement.

Generating settlement result for each settlement item and providing detail of calculation.
